# 字符大战几何体 / Characters vs Geometry

一个使用 TypeScript + Phaser 3 制作的极简黑白风五路选卡塔防 demo。玩家部署字符单位，对抗圆、三角、正方形和立方体 Boss 等几何敌人。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/`。项目已配置 GitHub Pages workflow，推送到 `main` 后可通过 GitHub Actions 自动部署。
